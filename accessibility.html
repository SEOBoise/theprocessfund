Commitment to Accessibility

The Process Fund is committed to ensuring digital accessibility for people with disabilities. We strive to continually improve the user experience for everyone and apply relevant accessibility standards.

Measures to Support Accessibility

To ensure accessibility, The Process Fund:

  • Uses semantic HTML and proper heading structure;
  • Provides alternative text for all meaningful images;
  • Ensures sufficient color contrast between text and backgrounds;
  • Makes all functionality available from a keyboard;
  • Tests our pages with popular screen readers;
  • Conducts periodic manual audits and automated scans to identify and fix issues.

Conformance Status

The Web Content Accessibility Guidelines (WCAG) define requirements for designers and developers to improve accessibility for people with disabilities. The Process Fund endeavors to conform with WCAG 2.1 Level AA.

Feedback

We welcome your feedback on the accessibility of our site. If you encounter accessibility barriers, please let us know so we can address them promptly:

  • Email: info@Theprocessfund.com

Compatibility with Assistive Technology

The Process Fund is designed to be compatible with modern browsers (Chrome, Firefox, Safari, Edge) and assistive technologies such as JAWS, NVDA, and VoiceOver.

Technical Specifications

Accessibility of The Process Fund relies on the following technologies to work with the particular combination of web browser and any assistive technologies or plugins installed on your computer:

  • HTML
  • WAI-ARIA
  • CSS
  • JavaScript

Assessment Approach

The Process Fund assesses the accessibility of the site through:

  • Automated testing tools;
  • Manual testing using screen readers and keyboard navigation;
  • User feedback.
